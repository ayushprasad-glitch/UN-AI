A simple Node.js app that lets you connect to multiple AI models through **one unified API**.

Supported Providers:
- **OpenAI GPT-5**
- **Google Gemini 3**
- **xAI Grok**
- **Perplexity**
- **Meta Llama**
- **GitHub Copilot (stub)**

This project organizes each model into its own provider file and exposes a single `/api/chat` endpoint for sending messages to any provider.
